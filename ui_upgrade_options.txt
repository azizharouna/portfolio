UI Upgrade Options:

1. Modern Minimalism:
	* Simplify existing styles and layout
	* Use a clean and modern color scheme
	* Remove unnecessary elements

2. Bold and Bright:
	* Introduce bold typography
	* Use vibrant colors to create a more energetic look
	* Add animations and transitions to enhance the user experience

3. Material Design:
	* Incorporate Material Design principles
	* Use a consistent color scheme and typography
	* Add depth and dimension to the UI

4. Dark Mode:
	* Add a dark mode option
	* Use a dark color scheme and adjust typography accordingly
	* Provide a toggle to switch between light and dark modes
